# Server Performance Stats

A Bash script that analyzes basic Linux server performance statistics.

## Description

This project contains a shell script that collects and displays important server information such as CPU usage, memory usage, disk usage, and running processes.

The script is designed to run on any Linux server.

## Features

- Display CPU usage
- Display memory usage
- Display disk usage
- Display running processes
- Simple Linux system monitoring using Bash commands

## Requirements

- Linux environment
- Bash shell
