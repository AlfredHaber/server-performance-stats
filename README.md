# Server Performance Stats

A Bash script that analyzes basic Linux server performance statistics.

## Description

This project contains a shell script (`server-stats.sh`) that collects and displays important server information such as CPU usage, memory usage, disk usage, and running processes.

The script is designed to run on any Linux server and provides a simple way to monitor basic system performance.

## Features

- Display CPU usage
- Display memory usage (used and free memory)
- Display disk usage
- Display running processes
- Display top system processes
- Simple Linux system monitoring using Bash commands

## Requirements

- Linux environment
- Bash shell
- WSL, Linux VM, or a Linux server
